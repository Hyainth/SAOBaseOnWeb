# SAOBaseOnWeb
《刀剑神域》当前深受大家的喜爱，《刀剑神域》自2010年起从未消失在年度作品的排名中，在2012和2013年更是榜上之首，而在网络上，《刀剑神域》的网上介绍也是处于一片半空白的状态，所以，在各种原因的促使下，本小组成员制定了《刀剑神域》的赏析网站，使用户进一步了解《刀剑神域》这部作品。
